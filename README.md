# YouTube Data Extraction Toolkit

A collection of lightweight web-based tools for extracting and analyzing YouTube data using the YouTube Data API v3.

## Included Tools

### 1. [YouTube Comment Extractor](./Tools%20collect%20comments.html)
- Extract comments from one or multiple YouTube channels.
- Retrieve channel names and video URLs.
- Export results in a table format for Excel or AI analysis.
- Bilingual interface (English / Vietnamese).

### 2. [YouTube Channel Tracker](./Youtube%20Channel%20Tracker.html)
- Track and analyze key metrics of multiple YouTube channels simultaneously.
- Monitor subscriber counts and total views.
- Detect channel country/region.
- Calculate average video duration for videos uploaded in the last 30 days.
- Analyze video posting frequency (videos per week).
- Support for @handle, Channel ID, or channel name input.

### 3. [YouTube Tag Extractor](./Youtube%20Tag%20Tracker.html)
- Extract tags from YouTube videos.
- Discover the most common tags from a channel or keyword search.
- Generate SEO-friendly keyword lists.
- Supports English and Vietnamese.

### 4. [YouTube Keyword Research Tool](./Keyword%20Research.html)
- Research keywords for SEO and content planning.
- Supports English and Vietnamese.

### 5. [YouTube API Setup Guide](./API%20YOUTUBE.txt)
- Step-by-step instructions for creating a YouTube Data API Key.
- English and Vietnamese documentation.
- Quick start guide for beginners.

## Features
- Built with HTML, CSS, and Vanilla JavaScript
- Uses YouTube Data API v3
- No backend required
- Responsive and lightweight
- Bilingual UI (EN / VI)
- Optimized for SEO research and data analysis
- Works directly in your browser

## Requirements
- A valid YouTube Data API v3 Key
- Modern web browser (Chrome, Firefox, Safari, Edge)

## How to Get Started

1. **Get API Key**: Visit [YouTube Data API Setup Guide](./API%20YOUTUBE.txt)
2. **Access Tools**: Visit [https://longk20.github.io/YOUTUBE-SEO-SUPPORT-TOOLS/](https://longk20.github.io/YOUTUBE-SEO-SUPPORT-TOOLS/)
3. **Use the Tools**: Select a tool, enter your API key, and start analyzing YouTube data

## Supported Input Formats

- **Channel Handle**: @channelname
- **Channel ID**: UC_xxxxxxxxxxxxx
- **Channel Name**: Text search (auto-detected)
- **Video URL**: Full YouTube video links
- **Keywords**: Text keywords for search
