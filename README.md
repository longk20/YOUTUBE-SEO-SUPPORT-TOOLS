# YouTube Data Extraction Toolkit

A collection of lightweight web-based tools for extracting and analyzing YouTube data using the YouTube Data API v3.

## Included Tools

### 1. [YouTube Comment Extractor](./Tools%20collect%20comments.html)
- Extract comments from one or multiple YouTube channels.
- Retrieve channel names and video URLs.
- Export results in a table format for Excel or AI analysis.
- Bilingual interface (English / Vietnamese).

### 2. [YouTube Tag Extractor](./Youtube%20Tag%20Tracker.html)
- Extract tags from a specific video.
- Discover the most common tags from a channel or keyword search.
- Generate SEO-friendly keyword lists.
- Supports English and Vietnamese.

### 3. [YouTube Keyword Research Tool](./Keyword%20Research.html)
- Research keywords for SEO and content planning.
- Supports English and Vietnamese.

### 4. [YouTube API Setup Guide](./API%20YOUTUBE.txt)
- Step-by-step instructions for creating a YouTube Data API Key.
- English and Vietnamese documentation.
- Quick start guide for beginners.

## Features

- Built with HTML, CSS, and Vanilla JavaScript
- Uses YouTube Data API v3
- No backend required
- Responsive and lightweight
- Bilingual UI (EN / VI)
- Optimized for SEO research and data analysis

## Requirements

- A valid YouTube Data API v3 Key
- Modern web browser
